# FCU-1101-Object Oriented Design
物件導向設計

## Environment
1. Install [Java SE JDK](https://www.oracle.com/java/technologies/downloads/).

2. Compile Java program.

    ```
    javac Main.java
    ```

3. Run Java program.
  
    ```
    java Main.java
    ```

4. （Select）Install [Eclipse](https://www.eclipse.org/downloads/).
